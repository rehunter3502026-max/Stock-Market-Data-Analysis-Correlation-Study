# Stock-Market-Data-Analysis-Correlation-Study

This project performs exploratory data analysis (EDA) on multiple stocks and the S&P 500 index to analyze price trends, daily returns, and correlation structures between assets.

It focuses on statistical insights and financial visualization using Python.

🚀 Key Features

Raw stock price visualization (static + interactive)

Daily return computation for all stocks

Correlation matrix of returns

Heatmap visualization of inter-stock relationships

Histogram analysis of return distributions

Market benchmark comparison (S&P 500)

🛠️ Tech Stack

Python

Pandas

NumPy

Matplotlib

Seaborn

Plotly

📈 Methodology
1️⃣ Data Preparation

Loaded historical stock dataset

Sorted data by date

Checked for missing values

2️⃣ Price Analysis

Plotted raw stock prices

Created interactive multi-stock comparison charts


4️⃣ Correlation Analysis

Calculated correlation matrix of daily returns

Visualized using heatmap

Identified positively and negatively correlated assets

5️⃣ Distribution Analysis

Plotted histograms of daily returns

Compared volatility across stocks

📊 Applications

Portfolio diversification analysis

Risk assessment

Preprocessing for CAPM or portfolio optimization

Quantitative finance research
